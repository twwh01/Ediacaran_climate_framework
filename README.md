# Supplementary code for Ediacaran coupling of climate and biosphere dynamics

If you are new to programming with R, first download and install RStudio. 

The code here is bundled as an R project which you can open and work with in RStudio. For this to work seemlessly, keep the directory structure as it is here. 

Running _dependencies.R_ will install all of the packages needed for this project. 

The scripts in directory "scripts" are labelled by the analysis and figures they produce.
